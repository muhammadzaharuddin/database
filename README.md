Mission 3 - Relational Database								
Recommended Duration								
0.5 to 1.0 day(s)								
								
Background								
In the previous mission, you should be storing your objects in a global list, which is not persisted. In Mission 3, you will need to persist your data in MySQL.								
								
Prerequisites								
1. Completed Mission 2								
2. Basic understanding about Relational Database								
3. Basic understanding about JDBC								
3. Basic understanding about MySQL								
4. Set up MySQL on your workstation. For Windows, you may use https://dev.mysql.com/downloads/installer/								
								
What you need to do								
1. Design your database table								
2. Create your table in your local MySQL instance								
3. Configure your Spring Boot application to connect to the MySQL instance using JDBC.								
4. Create a repository class to perform CRUD on the database table using JdbcTemplate. Try using @Repository annotation.								
5. Run your application, verify the CRUD endpoints using Postman. Remember to verify the records in the database too.								
6. Check in your source code in your GitHub repo.								
								
Resources								
1. JavaDoc for JdbcTemplate, https://docs.spring.io/spring-framework/docs/current/javadoc-api/org/springframework/jdbc/core/JdbcTemplate.html								
2. JDBC with Spring, https://www.baeldung.com/spring-jdbc-jdbctemplate
